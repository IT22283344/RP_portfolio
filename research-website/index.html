Project Overview

Abstract and research direction

Smart Fisher Lanka combines environmental data, AI analytics, and practical decision support to help coastal fishing crews plan safer and more productive trips.

Project Abstract

Sri Lanka's coastal fishing communities face shifting weather patterns, safety risks, and unpredictable catch conditions. This project proposes an AI-driven decision support system that merges sea-state intelligence, fishing habit insights, and predictive analytics into one practical interface.

Why it matters

The system aims to reduce avoidable risk, improve trip planning, and support sustainable fishing practices by giving crews and coordinators clear, timely recommendations they can trust.

Target Users

Coastal fishers, vessel captains, fisheries coordinators, and local support teams who need reliable decision guidance before heading to sea.

Research Outcome

A validated prototype that demonstrates practical value through predictions, alerts, and easy-to-read dashboards tailored to the fishing context.

Expected Impact

Safer trips, better fuel efficiency, improved catch strategy, and stronger awareness of environmental conditions around the coastline.

Key Features

Modules that support everyday decisions

Each module is built to give crews concise, actionable information instead of complex raw data.

Weather Prediction

Anticipates wind, rain, and rough-sea conditions so crews can choose safer departure windows and reduce weather-related risk.

Fish Classification

Supports species-level analysis using image or catch-data cues to help fishermen and planners understand what is likely to be available.

Safety Alerts

Highlights dangerous conditions such as storm fronts, strong currents, or high-risk travel periods before the crew departs.

Data Analytics

Turns historical fishing, weather, and trip data into easy-to-interpret insights that guide sustainable planning and review.

Technology Stack

Built with practical modern tools

The prototype combines a web interface, AI services, and connected data sources for a unified research showcase.

React

Used for component-based UI structure when extending the system into a richer interactive dashboard.

Node.js

Provides the server-side foundation for handling requests, APIs, and project data services.

Python

Supports AI model training, data cleaning, and research analytics for prediction and classification workflows.

TensorFlow

Implements machine learning models for forecast inference and image-based classification tasks.

IoT Sensors

Collect environmental and vessel readings that can feed the decision support pipeline in real time.
